# react-study.komiyamma.net



**React/Next.js** の初心者～中級者向けの学習教材を題材に、 
**Docusaurus** を採用し、Markdown を正とした SSG (Static Site Generation) 環境の構築ノウハウ検証を兼ねたリポジトリ。  

コンテンツ生成には **Antigravity** を活用。  
技術記事の執筆、アーキテクチャ図解、およびアセット生成のプロセスを AI中軸に据え、  
高品質なドキュメントを効率的(可能な限り自動的）にデリバリーするワークフローの実証を目的とする。  

**Docusaurus**自体がReactベースのフレームワークであるため、採用。
